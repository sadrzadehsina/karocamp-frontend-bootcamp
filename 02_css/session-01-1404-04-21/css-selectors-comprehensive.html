CSS Selectors & Combinators - Comprehensive Guide

Learn all types of CSS selectors with interactive examples

🎯 1. Basic Selectors

Element Selector

Selects all elements of a specific type. For example, all paragraphs, all headings, etc.

/* Selects all paragraphs */
p {
    color: #333;
    margin-bottom: 15px;
}

/* Selects all h2 elements */
h2 {
    color: #34495e;
    border-left: 4px solid #e74c3c;
}

This paragraph is styled using the element selector p.

All paragraphs will have the same styling applied.

Class Selector (.class)

Selects elements with a specific class attribute. Classes can be applied to multiple elements.

/* Selects elements with class="highlight" */
.highlight {
    background-color: #f1c40f;
    padding: 5px;
    border-radius: 3px;
}

.important {
    color: #e74c3c;
    font-weight: bold;
}

This text has a highlighted section and an important note.

This entire paragraph is marked as important.

ID Selector (#id)

Selects a single element with a specific ID. IDs should be unique on a page.

/* Selects the element with id="special-paragraph" */
#special-paragraph {
    background-color: #d4edda;
    border: 1px solid #c3e6cb;
    color: #155724;
    padding: 15px;
}

This paragraph has a unique ID and special styling that only applies to this element.

👥 2. Grouping Selectors

You can apply the same styles to multiple selectors by separating them with commas.

/* Applies font-family to all heading elements */
h1, h2, h3, h4, h5, h6 {
    font-family: 'Georgia', serif;
}

/* Multiple classes with similar styling */
.warning, .error, .alert {
    padding: 10px;
    border-radius: 5px;
    margin: 10px 0;
}
⚠️ This is a warning message
❌ This is an error message
ℹ️ This is an alert message

👨‍👩‍👧‍👦 3. Descendant & Child Selectors

Descendant Selector (space)

Selects all elements that are descendants (children, grandchildren, etc.) of a specified element.

/* Selects all p elements inside .parent-container */
.parent-container p {
    color: #8e44ad;
    font-style: italic;
}

/* Selects all span elements inside .parent-container */
.parent-container span {
    background-color: #f39c12;
    padding: 2px 5px;
}

Parent Container

This paragraph is inside the parent container and will be styled.

This nested paragraph is also a descendant and will be styled.

This span is highlighted

Direct Child Selector (>)

Selects only direct children (not grandchildren or deeper descendants).

/* Selects only direct li children of .direct-parent */
.direct-parent > li {
    background-color: #e8f5e8;
    border-left: 3px solid #27ae60;
}
  • Direct child - will be styled
    • Nested child - won't be styled by direct child selector
    • Another nested child - also not styled
  • Another direct child - will be styled
  • Third direct child - will be styled

✨ 4. Pseudo-classes

:hover Pseudo-class

Applies styles when a user hovers over an element.

.hover-demo:hover {
    background-color: #2980b9;
    transform: translateY(-2px);
    box-shadow: 0 4px 8px rgba(0,0,0,0.2);
}

:focus Pseudo-class

Applies styles when an element receives focus (like clicking on an input field).

.focus-demo:focus {
    border-color: #3498db;
    box-shadow: 0 0 0 3px rgba(52, 152, 219, 0.2);
}

:first-child and :last-child

Selects the first or last child element of its parent.

.child-demo li:first-child {
    background-color: #e8f8f5;
    border-left: 4px solid #16a085;
    font-weight: bold;
}

.child-demo li:last-child {
    background-color: #fdf2e9;
    border-left: 4px solid #e67e22;
    font-weight: bold;
}
  • First item (special styling)
  • Second item (normal styling)
  • Third item (normal styling)
  • Fourth item (normal styling)
  • Last item (special styling)

:nth-child

Selects elements based on their position. Great for creating striped tables or alternating patterns.

.nth-demo tr:nth-child(even) {
    background-color: #f8f9fa;
}

.nth-demo tr:nth-child(odd) {
    background-color: #ffffff;
}

🎯 5. Attribute Selectors

Basic Attribute Selector [attribute="value"]

Selects elements based on their attributes and values.

input[type="text"] {
    background-color: #e8f4fd;
    border: 2px solid #3498db;
}

input[type="email"] {
    background-color: #fff2e8;
    border: 2px solid #f39c12;
}

input[type="password"] {
    background-color: #fde8e8;
    border: 2px solid #e74c3c;
}

Advanced Attribute Selectors

/* Contains substring */
a[href*="example"] {
    color: #8e44ad;
    font-weight: bold;
}

/* Starts with */
a[href^="https"] {
    /* Secure link styling */
}

/* Ends with */
a[href$=".pdf"] {
    /* PDF link styling */
}

Links with different attribute patterns:

  • Secure example link (https + example)
  • Regular sample link
  • PDF document (https + .pdf)
  • Email to example domain

📚 Summary

You've learned about all the essential CSS selectors:

  • Element selectors: Target HTML elements directly
  • Class selectors (.class): Target elements with specific classes
  • ID selectors (#id): Target unique elements
  • Grouping selectors (,): Apply same styles to multiple selectors
  • Descendant selectors (space): Target nested elements
  • Child selectors (>): Target direct children only
  • Pseudo-classes (:hover, :focus, etc.): Target elements in specific states
  • Attribute selectors ([attr="value"]): Target elements by their attributes

🎉 Congratulations!

You now have a solid foundation in CSS selectors. Practice combining these selectors to create more specific and powerful styling rules!
