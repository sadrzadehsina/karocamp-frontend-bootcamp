# تمرین ۰۲: پروفایل ساده دانش‌آموز

## 🎯 اهداف یادگیری

با تکمیل این تمرین، شما تمرین خواهید کرد:

- ایجاد یک صفحه HTML متمرکز و تک موضوعی
- کار با عناصر قالب‌بندی متن پایه
- ساختاردهی محتوا با عناوین و پاراگراف‌ها
- استفاده مناسب از تأکید و متن پررنگ
- ساخت یک سند HTML ساده اما کامل

## 📋 توضیح تمرین

صفحه پروفایل دانش‌آموزی بسازید که اطلاعات پایه درباره یک دانش‌آموز را نمایش دهد. این تمرین روی سازماندهی اطلاعات شخصی به شکل واضح و ساختارمند با استفاده از عناصر بنیادی HTML تمرکز دارد.

## 🔧 الزامات

پروفایل دانش‌آموز شما باید شامل موارد زیر باشد:

1. **ساختار سند**
   - ساختار کامل سند HTML5
   - عنوان توصیفی صفحه شامل نام دانش‌آموز
   - بخش‌های مناسب `<head>` و `<body>`

2. **بخش‌های محتوا**
   - **نام دانش‌آموز**: عنوان اصلی با نام کامل دانش‌آموز
   - **اطلاعات پایه**: سطح کلاس و نام مدرسه با برچسب‌های پررنگ
   - **معرفی**: پاراگراف معرفی شخصی
   - **دروس**: لیست دروس مورد علاقه یا فعلی
   - **اهداف**: آرزوهای آینده و اهداف شغلی

3. **عناصر HTML برای تمرین**
   - `<h1>` برای نام اصلی دانش‌آموز
   - `<h2>` برای عناوین بخش‌ها
   - `<p>` برای محتوای متنی و اطلاعات
   - `<strong>` برای برچسب‌ها و اطلاعات مهم
   - `<em>` برای تأکید روی کلمات خاص (مانند اهداف شغلی)
   - `<hr>` برای جداسازی بخش‌های محتوا
   - `<br>` برای شکست خط در لیست‌ها

## 📝 دستورالعمل گام به گام

1. **راه‌اندازی سند**
   - یک فایل HTML جدید بسازید
   - عنوانی مانند "پروفایل دانش‌آموز: [نام]" اضافه کنید

2. **اضافه کردن عنوان اصلی**
   - از نام دانش‌آموز به عنوان عنوان اولیه `<h1>` استفاده کنید

3. **شامل کردن اطلاعات پایه**
   - سطح کلاس و مدرسه را با برچسب‌گذاری مناسب اضافه کنید
   - از `<strong>` برای برچسب‌ها استفاده کنید (کلاس:، مدرسه:)

4. **نوشتن معرفی**
   - پاراگرافی برای معرفی دانش‌آموز بسازید
   - ویژگی‌های شخصیتی و علایق را شامل کنید

5. **لیست دروس مورد علاقه**
   - از شکست خط (`<br>`) برای جداسازی هر درس استفاده کنید
   - لیست را ساده و خوانا نگه دارید

6. **اضافه کردن اهداف آینده**
   - درباره آرزوهای شغلی بنویسید
   - از `<em>` برای تأکید روی اصطلاحات مهم شغلی استفاده کنید

## 💡 پیشنهادات محتوا

- **نام دانش‌آموز**: از نام واقعی یا خیالی دانش‌آموز استفاده کنید
- **کلاس**: هر سطح کلاسی (مثلاً ۱۰، ۱۱، ۱۲، یا سال دانشگاه)
- **مدرسه**: نام مدرسه واقعی یا خیالی
- **معرفی**: توصیف کوتاه شخصیت و علایق اصلی
- **دروس**: دروس تحصیلی که دانش‌آموز از آن‌ها لذت می‌برد
- **اهداف**: آرزوهای شغلی، برنامه‌های دانشگاهی، یا اهداف شخصی

## 🎨 ساختار نمونه

```text
نام دانش‌آموز (H1)
کلاس: [سطح] (با برچسب پررنگ)
مدرسه: [نام مدرسه] (با برچسب پررنگ)

معرفی (H2)
[پاراگراف معرفی شخصی]

دروسی که دوست دارم (H2)
[لیست دروس جدا شده با شکست خط]

اهداف (H2)
[پاراگراف درباره آرزوهای آینده با تأکید روی اصطلاحات شغلی]
```

## 🔍 نواحی تمرکز

- **سلسله‌مراتب واضح اطلاعات**: از عناوین برای سازماندهی محتوا استفاده کنید
- **تأکید مناسب متن**: از `<strong>` برای برچسب‌ها، `<em>` برای تأکید شغلی استفاده کنید
- **ساختار تمیز**: بخش‌ها را با خطوط افقی جدا کنید
- **ساختار کامل HTML**: چارچوب اولیه سند را فراموش نکنید
