CSS Flexbox Complete Guide

1. Introduction to Flexbox

Flexbox (Flexible Box Layout) is a powerful CSS layout method that provides an efficient way to arrange, distribute, and align items in a container, even when their size is unknown or dynamic.

.container { display: flex; /* This makes the container a flex container */ }

9. Flexbox Cheat Sheet

Container Properties:

/* Make element a flex container */ display: flex; /* Direction of items */ flex-direction: row | row-reverse | column | column-reverse; /* Wrapping behavior */ flex-wrap: nowrap | wrap | wrap-reverse; /* Shorthand for direction and wrap */ flex-flow: <direction> <wrap>; /* Horizontal alignment */ justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly; /* Vertical alignment */ align-items: stretch | flex-start | flex-end | center | baseline; /* Align wrapped lines */ align-content: stretch | flex-start | flex-end | center | space-between | space-around;

Item Properties:

/* How much item should grow */ flex-grow: <number>; /* How much item should shrink */ flex-shrink: <number>; /* Initial size before growing/shrinking */ flex-basis: <length> | auto; /* Shorthand for grow, shrink, basis */ flex: <grow> <shrink> <basis>; /* Override container's align-items for this item */ align-self: auto | flex-start | flex-end | center | baseline | stretch; /* Change visual order without affecting HTML */ order: <number>;
