4. Media Query Breakpoints Used:

  • Small Mobile: up to 479px
  • Large Mobile: 480px - 767px
  • Tablet: 768px - 1023px
  • Desktop: 1024px - 1199px
  • Large Desktop: 1200px and up

6. Tips for Teaching Media Queries:

  1. Always start with mobile-first approach
  2. Use the browser's developer tools to test different screen sizes
  3. Test on real devices when possible
  4. Don't forget the viewport meta tag!
  5. Consider touch interactions for mobile devices
  6. Performance matters - optimize images and fonts for mobile
